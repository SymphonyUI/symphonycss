What is Symphony Anthology?

Symphony Anthology is a semantic-first, framework-agnostic CSS framework designed for the AI era. It prioritizes semantic HTML and uses modern CSS features like container queries, cascade layers, and OKLCH colors.

How do I customize the theme?

Override CSS custom properties in your stylesheet. All design tokens use the --symphony-* prefix and can be easily customized.

:root {
  --symphony-primary: oklch(50% 0.2 280);
  --symphony-radius-md: 0.75rem;
}
Is it accessible?

Yes! Symphony is built with accessibility in mind, following WCAG 2.1 AA guidelines. It includes proper focus states, keyboard navigation, and screen reader support.
