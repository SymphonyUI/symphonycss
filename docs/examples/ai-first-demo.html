Frequently Asked Questions

What is AI-First CSS?

AI-First CSS means designing components that are self-descriptive to AI systems. Instead of utility classes that describe appearance, Symphony uses semantic elements and attributes that describe meaning and intent.
Is Symphony compatible with React/Vue?

Yes! Symphony CSS works with any framework. Symphony Elements (Web Components) have adapters for React, Vue, Svelte, and vanilla JS. They use standard Web Components that work everywhere.
What is NLWeb?

NLWeb (Natural Language Web) is a Microsoft initiative to make web pages queryable by natural language. Symphony components include nlweb-* attributes that make your pages compatible with this emerging standard.
