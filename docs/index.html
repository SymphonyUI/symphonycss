Getting Started

Symphony CSS is a modern, modular CSS framework built with cutting-edge CSS features.

Installation

npm install symphony-css

Usage

<link rel="stylesheet" href="path/to/symphony.css">

Key Features

  • CSS Cascade Layers - Predictable specificity control
  • CSS Custom Properties - Typed, animatable design tokens
  • Framework Agnostic - Works with React, Vue, Svelte, and vanilla HTML
  • BEM Naming - Clear, maintainable class names
  • Semantic First - Minimal classes, maximum semantics
